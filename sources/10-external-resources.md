## 1. Agent Skills Marketplace
**URL:** https://skillsmp.com/
Nearly 60,000 Claude Skills. Browse and install skills for Claude Code.

## 2. MCP Tools (Model Context Protocol)
**URL:** https://modelcontextprotocol.io/
Complete documentation for MCP - the protocol that lets Claude connect to external services (Slack, GitHub, databases, APIs, etc). Includes guides for developing and connecting local servers.

## 3. Claude Prompt Library
**URL:** https://platform.claude.com/docs/en/resources/prompt-library/library
Anthropic's official prompt library with ready-to-use prompts for various tasks.

- Data & Analysis (root-cause-tracing, CSV summarizer, postgres)
- Scientific & Research Tools (125+ scientific skills, deep research)
- Writing & Research (article-extractor, content-research-writer)
- Media & Content (youtube-transcript, video-downloader, image-enhancer)
- Security & Web Testing (OWASP, systematic-debugging, ffuf)
- Utility & Automation (file-organizer, skill-creator)
- Collaboration & Project Management (Linear, git-pushing, meeting insights)
